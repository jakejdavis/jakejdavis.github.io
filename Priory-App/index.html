Priory App

A webapp which collects infomation from SMHW and E-Praise and shows the data all in one place. Made for Priory School students, created with a Vue.js frontend and PHP backend.

Getting Started

If you solely want to use the webapp, the production version is found at https://jakejdavis.me/priory.

Hosting

Alternatively if you want to host the webapp, you can quickly deploy it to a PHP server where the only prerequisite is curl to make requests to E-Praise and SMHW.

Issue Reporting

If you find any issues with the webapp, please report it! This will increase the chances of it getting fixed sooner. Please follow the issue template when creating an issue.
